

L’obiettivo dei questo progetto è quello di sviluppare un prototipo di applicazione gestionale in C++ con in-
terfaccia testuale per un servizio di noleggio di biciclette elettriche (e-bike). L’applicazione deve
permettere all’utente di localizzare i mezzi più vicini, verificare l’autonomia residua e gestire il
processo di noleggio simulando tempi e costi.

Questo codice è stato scritto in c++ usando la piattarfoma online gbd.
